# Structurally accurate MAVLink Schema
A MAVLink v2 schema that is actually accurate to the binary struct of MAVLink v2 packets. <br>

These `XML` and `json` files have everything you need to accurately interpret and parse all MAVLink v2 packets. The exact file you choose to use will depend on your exact use case, but these files are intended to allow you to parse MAVLink without relying on the Black Box that is `pymavlink`.

## Repo Layout
* All `XML` and `json` files you need are in `./`.
* `/src` contains the code used to generate the `XML` and `json` files.
* `/assets` contain files used by scripts in `/src` to generate these `XML` and `json` files. Most of these are generated by <a href="https://github.com/KCGD">KCGD</a> in `dvdsh` while working together on Senior Design.
